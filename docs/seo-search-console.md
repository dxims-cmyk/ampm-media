## Search Console Setup Steps

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add property → URL prefix → `https://mediampm.com`
3. Verify via HTML tag method
4. Copy the verification code and add to `src/app/layout.tsx`:
   ```tsx
   // In metadata export
   verification: {
     google: 'YOUR_VERIFICATION_CODE_HERE',
   },
   ```
5. Deploy and verify
